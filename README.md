# Hyperspectral Beverage Classification

## Overview

This project classifies beverages using hyperspectral spectral data with Machine Learning and Deep Learning techniques.

## Best Model

- **Model:** Multi-Layer Perceptron (MLP)
- **Accuracy:** 92.12%

## Features

- Upload Excel (.xlsx) or CSV (.csv) files
- Predict beverage classes
- Display Top 3 predictions with confidence scores
- Download prediction results

## Technologies Used

- Python
- Streamlit
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy
- Joblib

- Papaya
- Coffee

- Orange
- Tea
- Wine
